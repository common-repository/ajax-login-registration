== Description ==
This plugin is useful to add user login and registration. Plugin use ajax for login and registration so user experience awesome. Plugin provide separate shortcodes for login and registration so developer can add these shortcodes in separate page.


== Installation ==
Please follow below instructions to install this plugin to your WordPress website.

1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally
2. Upload to your site
3. Navigate to wp-admin/plugins.php on your site (your WP plugin page)
4. Activate this plugin

Also you can just install it with WordPress Admin Panel by going to Plugins / Add New and type this Ajax Login Registration.
